### Events

| Prop                         |     Type     | Description                                            | Default |
| :--------------------------- | :----------: | ------------------------------------------------------ | :-----: |

| **`onRestore`**              | `{Function}` | Resets option event.                                   |  null   |
| **`onDataViewChanged`**      | `{Function}` | Changing event of data view tool in toolbox.           |  null   |
