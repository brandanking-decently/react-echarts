# Installation

In order to use `@kbox-labs/react-echarts`, all you need to do is install the npm package:

```sh
pnpm add @kbox-labs/react-echarts
```

## Usage

To start using `@kbox-labs/react-echarts`, you just need to import the **`<EChart />`** component from the root folder.

```js
import { EChart } from '@kbox-labs/react-echarts'

function MyChart() {
  return (
    <EChart
      xAxis={{
        type: 'category'
      }}
      yAxis={{
        type: 'value',
      }}
      series={[
        {
          type: 'line',
          data: [
            ['2022-10-17', 300],
            ['2022-10-18', 100]
          ]
        }
      ]}
    />
  )
}
