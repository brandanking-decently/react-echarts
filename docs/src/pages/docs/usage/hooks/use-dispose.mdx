# useDispose

Destroys chart instance, after which the instance cannot be used any more.

## Usage

```js
import { useDispose } from '@kbox-labs/react-echarts';

function App() {
  useDispose({ target: 'group1' }, [...deps]);

  return (
    ...
  )
}
```

## API

```typescript
type UseDispose = (
  options: {
    target: ECharts | HTMLDivElement | HTMLCanvasElement
    enabled?: boolean
  },
  deps: unknown[]
) => void
```

### Parameters

- **`target`**: Target to be destroyed.
- **`enabled`**: Whether the hook should call the `disconnect` function or not.
  Defaults to `true`.
- **`deps`**: Array of reactive values that trigger the hook.
