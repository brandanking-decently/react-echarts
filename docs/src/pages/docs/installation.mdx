# Installation

In order to use `@kbox-labs/react-echarts`, all you need to do is install the
npm package:

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tabs.Tab>
  ```bash
  pnpm add echarts @kbox-labs/react-echarts
  ```
  </Tabs.Tab>
  <Tabs.Tab>
  ```bash
  npm i echarts @kbox-labs/react-echarts
  ```
  </Tabs.Tab>
  <Tabs.Tab>
  ```bash
  yarn add echarts @kbox-labs/react-echarts
  ```
  </Tabs.Tab>

</Tabs>
