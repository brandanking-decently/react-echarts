# useConnect

Connects interaction of multiple chart series.

## Usage

```js
import { useConnect } from '@kbox-labs/react-echarts';

function App() {
  useConnect({ group: 'group1', enabled: true }, []);

  return (
    ...
  )
}
```

## API

```typescript
type UseConnect = (
  options: { group: string | Array; enabled?: boolean },
  deps: unknown[]
) => void
```

### Parameters

- **`group`**: Group id, or array of chart instance.
- **`enabled`**: Whether the hook should call the `connect` function or not.
  Defaults to `true`.
- **`deps`**: Array of reactive values that trigger the hook.
