# useDisconnect

Disconnects interaction of multiple chart series. To have one single instance to
be removed, you can set `group` of chart instance to be null.

## Usage

```js
import { useDisconnect } from '@kbox-labs/react-echarts';

function App() {
  useDisconnect({ group: 'group1', enabled: true }, []);

  return (
    ...
  )
}
```

## API

```typescript
type UseDisconnect = (
  options: { group: string; enabled?: boolean },
  deps: unknown[]
) => void
```

### Parameters

- **`group`**: Group id.
- **`enabled`**: Whether the hook should call the `disconnect` function or not.
  Defaults to `true`.
- **`deps`**: Array of reactive values that trigger the hook.
