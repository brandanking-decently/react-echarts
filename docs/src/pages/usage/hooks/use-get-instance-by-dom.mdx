# useGetInstanceByDom

Returns chart instance of dom container.

## Usage

```js
import { useGetInstanceByDom } from '@kbox-labs/react-echarts';

function App() {
  const instance = useGetInstanceByDom({
    target: 'group1'
    }, [...deps]);

  return (
    ...
  )
}
```

## API

```typescript
type UseGetInstanceByDom = (
  target: HTMLDivElement | HTMLCanvasElement,
  deps: unknown[]
) => ECharts | undefined
```

### Parameters

- **`target`**: Target to be destroyed.
- **`enabled`**: Whether the hook should call the `disconnect` function or not.
  Defaults to `true`.
- **`deps`**: Array of reactive values that trigger the hook.
